# cqrs-bank-account-demo




## 背景和目的
### 1.学习EventSourcing + CQRS + DDD的架构思想
### 2.测试Axon重播（replay）功能




## 心得和体会

### 优势
* 方便进行溯源与历史重现
* 方便bug修复
* 提供非常好性能
* 便于事件分析


### 适用场景
* 不适用于大量CRUD或者多变需求的场景
* 业务复杂但稳定。如收益计算模型、订单管理
* 过程（溯源）比结果更重要
* 版本管理、版本回退


### 挑战
* 思维方式的转变。传统开发模式转变领域驱动设计
* 适用场景的局限
* 分布式事务的问题
